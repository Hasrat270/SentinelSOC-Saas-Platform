# Contributing to SentinelSOC 🛡️

We love contributions! SentinelSOC is an open-source project licensed under the **AGPL-3.0**, which means we value transparency and community collaboration.

## How Can I Contribute?

### 🛡️ Improve the Security Agent
You can help by adding new threat detection patterns to the `sentinel-soc-agent`. 
- SQL Injection patterns
- New XSS payloads
- Rate limiting features

### 📊 Dashboard Enhancement
Help us make the SOC dashboard even more powerful by adding:
- New visualization charts
- Interactive log filtering
- Dark mode refinements

### 🐛 Bug Reports
If you find a bug, please open an issue with:
1. A clear description of the problem.
2. Steps to reproduce.
3. Your environment details.
